# Smart Water Kyzylorda v2.0
## Интеллектуальная система учета водных ресурсов

Проект предназначен для автоматизации сбора показаний с водомеров и мониторинга состояния водопроводной сети в режиме реального времени.

### 🚀 Основные возможности
- **Автоматическая передача данных**: Интеграция с Arduino/ESP32 для передачи показаний по MQTT/HTTP.
- **Мониторинг аномалий**: Детекция протечек и попыток несанкционированного воздействия (магниты, скручивание).
- **Биллинг**: Автоматический расчет стоимости потребленной воды согласно тарифам.
- **История**: Визуализация потребления за неделю/месяц.

### 🛠 Технический стек
- **Frontend**: HTML5, Vanilla CSS, JavaScript (Chart.js).
- **Backend (MCU)**: Arduino Nano / NodeMCU.
- **Сенсоры**: Датчик потока (YF-S201), Датчик давления, Датчик Холла (детектор магнитов).

### 📂 Структура проекта
- `index.html` — Панель управления (Дашборд).
- `readme.md` — Данная документация.

### 📈 Логика безопасности
Система отслеживает вектор изменения показаний. Если зафиксировано *уменьшение* общего объема (попытка скручивания), активируется режим `ALARM` и отправляется уведомление в Telegram.
